# Weather WebApp

A simple and responsive web application that displays real-time weather information for any city. Built using **HTML**, **CSS**, and **JavaScript**, the app fetches accurate live data from a weather API and shows key details such as temperature, humidity, wind speed, and sky conditions in a clean interface.

## Features

* Real-time weather data
* Search any city worldwide
* Displays temperature, humidity, wind speed, and weather conditions
* Clean and minimal UI
* Works on desktop and mobile

## Technologies Used

* HTML
* CSS
* JavaScript
* Weather API (https://open-meteo.com/)

## How to Use

1. Enter a city name in the search box.
2. Press the search button.
3. View the live weather details instantly.

## Setup

1. Clone this repository:

   ```
   git clone https://github.com/Suvesh108/WeatherNow.git
   ```
2. Open the project folder.
3. Open `index.html` in your browser.
4. Add your API key in the JavaScript file (if required).
